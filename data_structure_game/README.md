# Data Structure Puzzle Game

An educational game that helps users learn and practice data structures through interactive puzzles.

## Features

- Multiple data structures to learn (Stack, Queue, Linked List, Binary Tree, Graph)
- Progressive difficulty levels
- Interactive visualizations
- Score tracking
- Educational feedback

## Installation

1. Make sure you have Python 3.8+ installed
2. Clone this repository
3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## How to Run

```
python main.py
```

## Controls

- Use arrow keys to navigate
- Press ENTER to select
- Press ESC to go back/exit

## Game Flow

1. Start Game
2. User Authentication
3. Main Menu
4. Data Structure Selection
5. Level Selection
6. Puzzle Solving
7. Feedback & Learning
8. Score Update
9. Next Level / Exit

## Project Structure

- `main.py` - Main game loop and state management
- `game/` - Game modules
  - `core/` - Core game logic
  - `ui/` - User interface components
  - `states/` - Game state implementations
- `assets/` - Game assets (images, sounds, etc.)
- `levels/` - Level definitions
- `utils/` - Utility functions
- `tests/` - Test files
